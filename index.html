Мои работы

EIKON financial.thomsonreuters.com

Последние полгода я работаю над различными модулями для продукта Thomson Reuters - Eikon. По сути, это множество приложений для отображения различной финансовой информации. Каждое приложение независимо и различается по реализации. Большая часть модулей, с которыми я работал базируется на Angular 1.5.
